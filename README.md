# ROCK, PAPER, SIMULATOR!
This was a game I made around 2022. It used to have a *lot* more content, and menus where you could add different items, backgrounds, and music! But when I simplified the stuff down to be used in my portfolio, I got paranoid about professionalism/simplicity and removed everything else besides the gameplay. It's unfortunate, because it used to be a cool example of loading in custom content and displaying it in a game.

## WHAT IT USED TO BE
- A set of menus, letting you select "what beats what", how many of each item is spawned in, what background should be used, and what music should play.
- There were around 10 different items to select from, with the latter 7 being the faces of classmates in my CS class. 
- The game then runs, and everything moves in a defined way. Rock, Paper, and Scissors all move in the same bouncing position while the students had uniquely programmed movements. 
- If too much time progressed, the words "HURRY UP" would flash, and bombs would appear to take out letters en masse, and skulls would appear that killed everything that touched it.
- The game would end, and give you the option to either replay or go back to the menu.

## WHAT WE HAVE NOW
- The game starts with rock, paper, and scissors by default.
- No music plays.
- The options are changed to QUIT or RESTART at the end.

## TAKEAWAYS
- Sure, this is the more optimized and streamlined choice, but it's quite sad to see something with personality get muzzled like this.
- I will make sure not to do this again, but I don't have time to change it back (especially as I don't have version control for the deleted code :c).
